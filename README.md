# Agentic Control Plane Builder

A standalone React/Vite application for designing an agentic control plane before building AI agents.

## What it builds

- Agent team configuration
- Tool/app connector map
- Memory vault scope
- Workflow orchestration plan
- Human approval and governance rules
- Dashboard/output plan
- Generated implementation blueprint

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

- Framework: Vite
- Install: `npm install`
- Build: `npm run build`
- Output: `dist`
